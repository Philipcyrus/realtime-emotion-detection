## [0.1.0] - 2024-06-14

### Added

- Added real-time webcam emotion detection with OpenCV.
- Added Keras/TensorFlow model loading, training, and evaluation commands.
- Added still-image prediction and privacy-safe JSON/HTML reporting.
- Added demo-data generation and FER2013-style dataset import tooling.
- Added project scripts for setup checks, live runs, training, evaluation, and smoke tests.
- Added unit tests covering configuration, datasets, model behavior, reporting, and core data models.
- Added repository hygiene for GitHub publishing, including ignored local data, reports, model binaries, virtual environments, and local tool metadata.
